README 작성법
======================
readme는 마크다운 문법을 이용하여 작성한다. 마크다운은 특수기호와 문자를 이용한 간단한 구조의 문법으로 직관적으로 인식할 수 있는 언어다. 이를 통해서 설치방법, 소스코드 설명, 이슈 등을 간단하게 기록할 수 있다. 
## 마크다운 문법
### 1. 제목(Headers)
제목을 표시할 때는 `#`을 이용해 나타낸다. `<h1>`부터 `<h6>`까지 나타낼 수 있다.  
(예시)
```
# h1
## h2
### h3
#### h4
##### h5
###### h6
```
(실행결과)  
# h1
## h2
### h3
#### h4
##### h5
###### h6
### 2. 인용문(BlockQuotes)
인용문은 `>`를 이용해 나타낸다. 중첩된 인용문도 만들어낼 수 있다.  
(예시)
```
> 인용문
>> 중첩된 인용문
```
(실행결과)  
> 인용문
>> 중첩된 인용문
### 3. 표(Table)
헤더 셀을 구분할 때는 3개 이상의 `-`(hyphen/dash)가 필요하다.  
헤더 셀을 구분하면서 `:`(colons) 기호로 좌, 우, 가운데 정렬이 가능하다.  
항목끼리의 구분은 `|`(vertical bar) 로 한다.  
(예시)
```
 Header1 | Header2 | Header3 
 :------ |:-------:| -------:
 content1|content2 | content3
```
(실행결과)  
| Header1 | Header2 | Header3 |
| ------- |:-------:| --------|
| content1|content2 | content3|


### 4. 목록(Lists)
#### 순서있는 목록
숫자와 점(.)을 이용하여 나타낸다. 어떤 숫자를 입력해도 내림차순으로 정리되어 나타난다.  
(예시)
```
1. 첫번째 목록
1. 두번째 목록
1. 세번째 목록
```
(실행결과)  
1. 첫번째 목록
1. 두번째 목록
1. 세번째 목록

#### 순서없는 목록

`*`, `+`, `-` 로 나타낸다.  
(예시)
```
* list1
  * list 1-1
  * list 1-2
    * list 1-2-1
```
(실행결과)  
* list1
  * list 1-1
  * list 1-2
    * list 1-2-1

### 5. 코드블럭(Code Blocks)
첫 줄과 마지막 줄에 ``` 또는 `~~~`를 삽입한다.  
(예시)
```
~~~
이것은
코드블럭
입니다.
~~~
```
(실행결과)
~~~
이것은
코드블럭
입니다.
~~~
### 6. 인라인 코드(Inline Code Blocks)
`로 감싸진 텍스트.  
(예시)
```
`인라인 코드 블럭`
```
(실행결과)  
`인라인 코드 블럭`
### 7. 강조(Emphasis)
이탤릭체(italic) : 별표`*` 또는 언더바`_` 로 감싼 텍스트  
두껍게(bold) : `**` 또는 `__`로 감싼 텍스트  
취소선 : 물결`~~`로 감싼 텍스트  
밑줄 : `<u> </u>` 로 감싼 텍스트  
(예시)
```
*italic*  
**bold**  
~~strike out~~  
<u>underline</u>
```
(실행결과)  
*italic*  
**bold**  
~~strike out~~  
<u>underline</u>  
### 8. 수평선
`*` 또는 `-` 또는 `_`를 세 개 이상 입력하면 수평선이 만들어진다.  
(예시)
```
***
---
___
```
(실행결과)  
***
---
___
### 9. 줄바꿈
띄어쓰기 두번 혹은 `<br>`을 이용해 줄을 바꾼다.  
(예시)
```
줄바꿈
줄바꿈<br>
줄바꿈
```
(실행결과)  
줄바꿈
줄바꿈<br>
줄바꿈

### 10. 링크
#### 외부 링크
##### 인라인 링크
`[링크이름](url)` 형식으로 외부로 이어지는 링크를 만든다.  
(예시)
```
[Google](https://www.google.co.kr/)
```
(실행결과)  
[Google](https://www.google.co.kr/)
##### 참조 링크
(예시)
```
[Google][1]
[Naver][2]
[1]:https://www.google.co.kr/
[2]:https://www.naver.com/
```
(실행결과)  
[Google][1]  
[Naver][2]  
[1]:https://www.google.co.kr/  
[2]:https://www.naver.com/  
##### URL 링크
(예시)  
```
https://www.google.co.kr/ 
```
(실행결과)  
https://www.google.co.kr/  
#### 내부 링크

### 11. 이미지
#### 인라인 이미지
(예시)
```
![alt text](/test.png)
```
(실행결과)  
![alt text](/test.png)
#### 링크 이미지
(예시)
```
![alt text](image_URL)
```
(실행결과)  
![alt text](image_URL)
#### 참조 이미지
(예시)
```
![alt text][1]
[1]: /test.png
```
(실행결과)  
![alt text][1]
[1]: /test.png
### 12. 이미지에 링크 달기
마크다운 이미지 코드를 링크 코드로 묶어 준다.  
(예시)
```
[![alt text](/googlelogo_color_272x92dp.png)](https://www.google.com/)
```
(실행결과)  
[![alt text](/googlelogo_color_272x92dp.png)](https://www.google.com/)
